SAP HANA Intro


SAP HANA On Demand for Premise and Cloud

SAP HANA, short for "High-Performance Analytic Appliance" is an in-memory, column-oriented, relational database management system developed by SAP. Data now resides in main-memory (RAM) and no longer on a hard disk. It is a combination of hardware and software made to process massive real time data using In-Memory computing. It combines row-based, column-based database technology. It's best suited for performing real-time analytics on massive data, and developing & deploying real-time applications.
